# GS Capture Documentation

GS Capture is a Blender addon for generating synthetic training data for 3D Gaussian Splatting and NeRF workflows.

This site is the user-facing documentation for installation, capture workflows, export formats, and training backend setup.

## What You Get

- Camera-synchronized image capture from selected objects
- Optional exports: COLMAP, `transforms.json`, depth, normals, masks
- Checkpoint/resume support for long capture runs
- Built-in training launchers for supported backends

## Recommended Versions

- Blender 4.5.1 LTS for stable production runs
- Blender 5.0 supported with compatibility updates
